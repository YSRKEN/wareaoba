戦場のカメラマン 青葉 Ver.1.5
※等幅フォント推奨。
--------------------------------------------------------------------------------
[ソフト紹介]
　艦これのスクショを撮る際、次のような経験はありませんか？
「スクショを撮るタイミングが遅れた！」
「他のウィンドウに邪魔されてしまった！」
「ブラウザがスクロールしてズレてしまった！」
　ですが、このソフトを使えばそんな心配は無用！
　ワンクリックですかさず連写し、ベストショットを逃しません！
　独自のリングバッファ実装により、画像保存ボタンを押す前の「過去」、
そして「未来」のスクショも保存することができます。自動撮影機能を利用すれば、
『対空カットイン』『昼戦カットイン』『昼戦連撃・昼戦砲撃』『夜戦カットイン』
『夜戦連撃・昼戦砲撃』『中破・大破時』『敵にダメージを与えた時』を判別して、
自動で保存することができます。
　800x480以外の画面でも、マニュアルモードに切り替えれば上記のリングバッファによる
保存機能を利用することが可能です！

[インストール・アンインストール方法]
・インストール方法
　zipファイルを解凍するだけです。
・アンインストール方法
　ファイルをそのまま削除するだけです。レジストリは使用しません。

[ファイル]
※必須ファイルは☆、そうでもないファイルは○が付いています。
☆wareaoba.exe……実行ファイル。
☆hspcv.dll……補助DLL。
○Readme.txt……この文章のことです。
○sourceフォルダ……ソースコードです。

[操作方法]
1. まず、「座標取得」ボタンを押します。これにより、艦これが起動しているブラウザ
   のハンドルと座標、および艦これの画面の相対座標を取得します。

2. 座標取得の際には、次の4つのモードが利用できます。
   ・オート……画面全体のスクショから検索。まずはこちらを試して下さい。
   ・クリック……画面全体から、艦これの画面をクリックして検索。
   ・クロップ……画面全体から、艦これの画面をマウスで囲って検索。
   ・マニュアル……任意の範囲を囲って、そこから画面を認識する。詳しくは後述

3. 座標取得に成功すると、画像保存ボタンが有効化されます。この状態で画像保存
   ボタンを押すと、決められた設定通りにスクショを連写し記録します。
   なお、画像のファイルパスはexeファイルと同じフォルダで、
   ファイル名はボタンを押した日時＋連番です。

4. 4つのコンボボックスによって、スクショを取る際の設定を変更できます。
   ・○枚……全体の撮影枚数を□とすると、□＝○×2＋1です。なぜこうなるかは後述。
   ・△fps……撮影時のfpsを指定します。5fpsなら0.2秒間隔で自動記録されます。

   ・bmp/png/jpeg……保存時の形式を指定します。保存速度はbmpが一番速いですが、
     容量を食うことを考えればpngやjpegにしておくべきでしょう。

   ・過去/現在/未来……「過去」だと「ボタンを押すまで」のスクショが保存されます。
     「現在」だと「ボタンを押してから○フレーム前〜○フレーム後」まで保存されます。
     「未来」だと「ボタンを押してから○×2フレーム後」まで保存されます。
     通常は「現在」モードで使うことが多いでしょう。

   ・ショートカットキー……チェックを入れると、Ctrl＋右クリックで保存できます。
     わざわざソフトの画像保存ボタンまでマウスを動かしたくない場合に有効です。

   ・自動撮影……チェックを入れると、特定のシーンが来た際に自動で保存できます。
     保存したいシーンについては、6.における「選択...」ボタンの説明をご覧ください。
     なお、2.で座標取得を「マニュアル」にして取得した際、当該画面のサイズが800x480以外
     だった場合はこの機能が使えません。

   ・ズレ補正……チェックを入れなくても、「対象ウィンドウが消滅した場合」と
     「対象ウィンドウのサイズが変わった場合」には反応します。チェックを入れると、
     「対象ウィンドウにおいてスクロールが行われた場合」に対応できます。ただ、最後のには
     誤爆のリスクが有りますので、ズレ補正が連発する状況ならOFFにしましょう。

5. 「参照...」ボタンからフォルダを選択すると、そのフォルダに撮影画像を保存できます。
   (デフォルトは3.にある通り、exeと同じフォルダに保存)

6. 「選択...」ボタンを押すと、自動撮影におけるシーンを選択できます。
   『対空カットイン』『昼戦カットイン』『昼戦連撃・昼戦砲撃』
   『夜戦カットイン』『夜戦連撃・昼戦砲撃』『中破・大破時』のそれぞれについて、
   チェックを入れていると自動保存します。ただ、後述する注意にもあるように、
   あるシーンの画像を保存している間は他のシーンの画像が保存できませんので悪しからず。

7. 2.で座標取得を「マニュアル」にして取得した場合、艦これの画面が800x480以外であっても……
   それどころか艦これ以外のあらゆる画面でも、リングバッファによるタイムシフト保存が可能です。
   もっとも800x480じゃないと自動撮影機能は使えませんが、それ以外では大いに役立つことでしょう。

8. ソフト紹介にもありますが、他ウィンドウに隠れても問題なく保存できます。
   また、ブラウザをスクロールしたりブラウザを動かしたりブラウザを終了したり
   しても、そのことを自動で検出し、座標を補正してくれます。
   ただ、ブラウザが終了した場合は流石に座標取得からやり直しですね。

[動作原理]
　ソフト紹介にもある通り、リングバッファに記録しています。要するに、一定間隔で
スクショを記録しておき、保存ボタンを押したらその中身を保存するわけです。
　撮影モードが「過去」の場合はそのまま保存すればいいのですが、「現在」の場合は
○フレーム後、「未来」の場合は○×2フレーム後にやっと保存を始めます。
　保存する際は、記録しておいたバッファからそのまま書き出すだけですので、ディスクの
速度に依らず確実に記録できます(書き出す間はソフトの動きも停止します)。

[注意]
　このソフトによって生じた損害(「バグで画像が保存できなかった」など)については保証できません。
　また、他ウィンドウに隠れても大丈夫なのはWindows 7・Windows 8.1で確認していますが、
WinAPIの仕様上、他のOSまで全く同じ挙動を示すかどうかは保証できません。
　そしてメモリについてですが、おおよそ800×480×3×(2×○＋1)バイトは消費します。
つまり、「99枚」にでもすると220MBほどのメモリを消費する上、保存時には199枚ものスクショが
保存されることになります。くれぐれもやり過ぎないようご注意下さい。
　また、「画像保存中は保存できない」仕様ですので、自動撮影機能を利用している場合
は特に注意して下さい。自動撮影機能において、オススメは「10枚」「5fps」です。
